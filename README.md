## 👨‍💻 About Me

I am a **Server System Network Manager** and a **Full-Stack Developer**, deeply passionate about building, securing, and managing scalable architectures. As the founder and owner of **[Blue Network](https://bluenetwork.tr/)**, I bridge the gap between robust software development and high-level system infrastructure.

* 🔭 I specialize in **Server Systems, Monitoring, and Virtualization Technologies** (ESXi, Proxmox).
* 🛡️ I have strong expertise in **Cybersecurity**, specifically in Web Application Security, vulnerability assessment, and ethical hacking.
* 💻 I develop comprehensive solutions using **Node.js, Express.js**, and various frontend technologies.
* ⚙️ Experienced in **System & Network Management**, ensuring optimal performance and absolute security.
